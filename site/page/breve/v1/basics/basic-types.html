Basic Types

Breve possesses a few basic types that map directly to Anima's underlying value types:

Type Description Default Value
signed or int Signed 64-bit integer. 0
unsigned or uint Unsigned 64-bit integer. 0
float or real 64-bit (double-precision) floating-point number. 0.0
string or str Text string. Empty string ("").
array or arr Dynamic list of values. Empty array ([]).
object or obj String-mapped collection of values. Empty object.
null or nil Null value. Indicates that the value does not exist. null
undefined or void Empty value. Different from null, as it indicates that the source itself does not exist whatsoever. void
